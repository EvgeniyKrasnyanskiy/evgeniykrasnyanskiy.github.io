# 🌲 Персональная визитка и Походный чеклист

Это исходный код моей личной страницы, доступной по адресу: 
👉 **[evgeniykrasnyanskiy.github.io](https://evgeniykrasnyanskiy.github.io)**

### 🛠 О проекте
Минималистичное веб-приложение, которое объединяет два моих интереса: программирование и отдых на природе.

*   **Визитка:** Краткая инфо и быстрый переход в Telegram.
*   **Чеклист:** Интерактивный список для сборов в поход (сохраняет состояние в `localStorage`).

### 💻 Технологии
*   **HTML5 / CSS3:** Адаптивная верстка, поддержка темной темы (автоматически подстраивается под систему).
*   **Vanilla JS:** Логика переключения страниц и работа с локальным хранилищем браузера.
*   **GitHub Pages:** Бесплатный хостинг прямо из этого репозитория.
